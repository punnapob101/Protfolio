## 📖 About the Project
This project is my personal portfolio website designed in a **Neo-Brutalism** style, blending modern typography with bold, high-contrast colors (e.g., solid black, white, and a vibrant neon green `#beff45`). It utilizes **Tailwind CSS** for layout and UI management to officially and attractively showcase my biography, technical skills, software projects, and activities.

## 🚀 Key Features
The website is divided into 4 main pages for easy navigation and comprehensive coverage:
1. 🏠 **Home (`index.html`):** A visually striking welcome page featuring a dynamic marquee text introduction.
2. 👤 **Profile & Skills (`one.html`):** Personal information, educational background, and Technical Skills (Java, C#, PHP, React, MySQL, etc.).
3. 💻 **Projects (`two.html`):** A collection of software development works, such as:
   - Boy Soccer Academy Evaluation System (AppSheet + React)
   - University Accident Notification System (PHP + MySQL)
   - AR Computer Hardware Learning Media (Unity)
   - RSA Algorithm Performance Optimization
4. 🏆 **Awards & Activities (`three.html`):** A showcase of achievements, certificates, or participation in various events.
5. 📞 **Contact (`four.html`):** Channels for communication and networking.

## 🛠️ Tech Stack
- **Structure & Styling:** Vanilla HTML5, CSS3
- **CSS Framework:** Tailwind CSS (via CDN)
- **Typography:** Google Fonts (Kanit)
- **Icons:** Lucide Icons
- **Scripting:** Vanilla JavaScript

## ⚙️ Installation & Run Guide
This project is a static website (HTML/CSS/JS) and can be easily launched without relying on a dedicated back-end server.
1. Clone or download the source code to your local machine.
2. Double-click the `index.html` file to open it directly in a web browser.
3. (Optional) If you are using VS Code, you can install the `Live Server` extension and click "Go Live" to see real-time updates as you edit the code.
